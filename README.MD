
可以在 Action 中下载 GitHub Workflow 构建的 jar 包。

## 功能

### Generate Codec/Stream Codec

生成 Minecraft Mod 所需的 Codec/Stream Codec。

- 支持 Record 类；
- 支持注解 Nullable；
- 支持 Optional, List, Map 类；
- 支持 Enum 枚举类。

### Generate AnvilLib Codec/Map Codec/Stream Codec

在必要时使用[铁砧库](https://github.com/Anvil-Dev/AnvilLib)生成 Minecraft Mod 所需的 Codec/Map Codec/Stream Codec。

- 支持 Record 类；
- 支持注解 Nullable；
- 支持 Optional, List, Map 类；
- 支持 Enum 枚举类（Map Codec 会自动转接到 Generate AnvilLib Codec）。
